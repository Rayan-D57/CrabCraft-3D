# Crab 3D Design 

This project is a 3D engineering design of a crab model developed using Onshape software. The design is based on a technique of assembling flat pieces to create a 3D structure.

## Tools and Techniques Used
* Onshape: For 3D modeling.

* Sketching: Using basic shapes and curves to draw the crab's body.

* Constraints: Applying geometric constraints to ensure the drawing's accuracy and convert it to a fully defined shape.

## Design Details
* Main Body: Designed using flowing curves to provide a natural shape.

* Legs and Claws: Designed as separate parts with slots measuring 20mm long and 10mm wide to facilitate assembly.

* Symmetry: The Mirror tool was used to ensure symmetry of the legs and claws on both sides of the body.
